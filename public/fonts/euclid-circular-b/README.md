# Optional local woff2

The docs site loads **Euclid Circular B** from ACKO CMS via `app/font-euclid.css` (woff2 URLs). You normally **do not** need files in this folder.

If you switch to self-hosted fonts later, add woff2s here and point `app/font-euclid.css` at `/fonts/euclid-circular-b/...` instead of the CDN.
